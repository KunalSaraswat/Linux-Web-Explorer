Project Summary: Text Processing with sed

This demo shows how to perform powerful text transformations using sed:

  • sed 's/old/new/' - Substitute text
  • sed '/pattern/d' - Delete lines matching pattern
  • sed 'Ni\text' - Insert text before line N
  • sed -n '2,4p' - Print specific lines (2-4)
  • sed 's/^/prefix/' - Add prefix to lines
  • sed -i.bak - In-place editing with backup
kunal@linux:~$ cat > employees.csv
Name,Department,Salary

kunal@linux:~$ awk -F ',' '{print $1}' employees.csv
Name
John
Sarah
Mike
Lisa
kunal@linux:~$ awk -F ',' '$3 > 80000 {print $1,$3}' employees.csv
Mike 82000
Lisa 90000
kunal@linux:~$ awk -F ',' 'BEGIN {print "High Earners:"} $3 > 70000 {print $1,$3}' employees.csv
High Earners:
John 75000
Sarah 68000
Mike 82000
Lisa 90000
kunal@linux:~$ awk -F ',' '{sum += $3; count++} END {print "Avg Salary:", sum/count}' employees.csv
Avg Salary: 78750
kunal@linux:~$ awk -F ',' '{print NR ". " $1}' employees.csv
1. Name
2. John
3. Sarah
4. Mike
5. Lisa
kunal@linux:~$ awk -F ',' '$1 ~ /^S/ {print $0}' employees.csv
Sarah,Marketing,68000

Project Summary: Data Processing with awk

This demo shows how to process and analyze structured data using awk:

  • awk -F ',' '{print $1}' - Print first column
  • awk '$3 > 80000' - Filter rows by condition
  • awk 'BEGIN {print "Header"} - Execute before processing
  • awk '{sum += $3} END {print sum}' - Calculate totals
  • awk '{print NR ". " $1}' - Add line numbers
  • awk '$1 ~ /^S/' - Pattern matching on fields
